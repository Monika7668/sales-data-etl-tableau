# 📈 Sales Data ETL & Tableau Analytics

## 📌 Business Problem

Sales data from CRM and ERP systems is fragmented, making it difficult to analyze performance.

---

## 🎯 Objective

Build an ETL pipeline and dashboard to provide actionable sales insights.

---

## 🛠 Tech Stack

* SQL
* Tableau
* ETL Pipelines

---

## 🔄 ETL Process

1. Extract data from multiple sources
2. Transform using SQL
3. Load into analytics-ready dataset

---

## 🧹 Data Quality Handling

* Removed duplicates
* Standardized region names
* Handled missing values

---

## 📊 Dashboard Features

* Revenue by region
* Product performance
* Interactive filters
* Heat maps

---

## 📊 Key Insights

* North region generated highest revenue
* Product B performed best overall
